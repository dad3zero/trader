Welcome to Star Trader 2090's documentation!
============================================

Star Trader 2090 is based on the game written by Dave Kaufman and information
about the game was published in `People's Computer Company` newsletter in 1974.

This project is based on true-grue (Peter Sovietov) attempt to port the game in
Python.
